CROSSOVER : untuk menghubungkan perangkat yang sama, contoh ("router ke router" "switch ke switch").

temen-temen bisa lihat sendiri yaa.. untuk Topologi sudah berwarna hijau semua ya, eitsss... tapi belum terhubung ke antar ROUTER yaa, terlebih dahulu kita harus settingg secara DHCP dulu ya, Pencet kembali di ROUTER 0 kita akan setting DHCP :

setting DHCP
ROUTER 0

Router>enable
Router#conf t
Enter configuration commands, one per line. End with CNTL/Z.
Router(config)#ip dhcp pool fa0/0
Router(dhcp-config)#default 192.168.2.1
Router(dhcp-config)#network 192.168.2.0 255.255.255.0
Router(dhcp-config)#dns 8.8.8.8
Router(dhcp-config)#ex
Router(config)#ip dhcp pool fa0/1
Router(dhcp-config)#default 192.168.3.1
Router(dhcp-config)#network 192.168.3.0 255.255.255.0
Router(dhcp-config)#dns 8.8.8.8
Router(dhcp-config)#ex
Router(config)#

setting DHCP
ROUTER 1

Setelah kalian sudah meyetting DHCP, langkah selanjutnya adalah kalian pergi ke PC untuk berikan IP yang sudah kita setting, lalu kalian pilh yang DHCP, maka IP addres akan muncul sendiri/keluar secara otomatis.
langkah TERAKHIR. yaitu kita "ip route" kan agar semua komputer dapat saling terhubung dengan antar Router.
ikuti cara di bawah ini :

setting IP route
ROUTER 0

Router>enable
Router#conf t
Enter configuration commands, one per line. End with CNTL/Z.
Router(config)#ip route 192.168.4.0 255.255.255.0 10.10.10.4
Router(config)#ip route 192.168.5.0 255.255.255.0 10.10.10.4
Router(config)#ip route 10.10.10.0 255.255.255.0 10.10.10.4
Router(config)#

setting IP route
ROUTER 1

Router>enable
Router#conf t
Enter configuration commands, one per line. End with CNTL/Z.
Router(config)#ip route 192.168.2.0 255.255.255.0 10.10.10.3
Router(config)#ip route 192.168.3.0 255.255.255.0 10.10.10.3
Router(config)#ip route 10.10.10.0 255.255.255.0 10.10.10.3
Router(config)#

Jika semua sudah di setting mulai dari ("pemberian IP", "setting DHCP", "setting IP route") langkah yang paling AKHIR ia lah, kalian saling "PING" kan antar router seperti Dibawah : 👍👌
